# Arcwell Game 2D (port from Android 5.0 to Android 15)
# -----NOTICE (For contributors)-----  
Supported architectures: armeabi-v7a, arm64-v8a, x86, x86_64;
-To build .apk of the game you need to download Android SDK/NDK command-line tools archive and unzip it in <C:\AndroidSDK> folder;  
-Also you need JDK 17.0.12 unzipped in <C:\jdk> folder;
-After that, compile the apk by starting the gradlew.bat file in cmd ("gradlew.bat clean assembleDebug").
