# Arcwell 2D Game
This game is in development. You can run it by cloning the whole repository and running the executable found in the Debug folder (Windows 10/11).
# ===== NOTICE (For contributors) =====
We use an Eclipse IDE, so this is a generic Eclipse C/C++ project.
-Make sure that you using the same IDE (also with C/C++ extension);
-Create a workspace in the Eclipse IDE;
-Clone the repository in the workspace folder (where the .metadata folder located);
-Import the ArcwellGame2D C/C++ project.
After all of that you can edit the project and push your commits in the "devel" branch.
